# Search

It is based on the [downshift](https://github.com/downshift-js/downshift) library.

## Default

<Playground>
  <Form initialValues={{}}>
    <ConnectedField
      component={Search}
      search={async function asyncSearch(s) {
        const response = await fetch(`http://www.omdbapi.com?apikey=41514363&s=${s}`)
        const data = await response.json()
        return data.Search
      }}
      label="Films"
      name="films"
      renderItem={item =>
        item && (
          <div style={{ display: 'flex' }}>
            <Shape width="50px" height="50px" shape="circle">
              <img src={item.Poster} />
            </Shape>
            {item.Title}
          </div>
        )
      }
      itemToString={item => item && item.Title}
    />
  </Form>
</Playground>
